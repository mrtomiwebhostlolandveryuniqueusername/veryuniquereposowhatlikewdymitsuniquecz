How to install Chrome Remote Desktop on Cloud Shell

So first, you need to open up Cloud Shell

After you opened it, follow these steps:

wget https://dl.google.com/linux/direct/chrome-remote-desktop_current_amd64.deb

That downloaded the Chrome Remote Desktop thing, so now we install it,

sudo apt install ./chrome-remote-desktop_current_amd64.deb

Now, we install XFCE (its possible it works with gnome too, i didnt try it)

sudo DEBIAN_FRONTEND=noninteractive \ apt install --assume-yes xfce4 desktop-base dbus-x11 xscreensaver

Okay, so now, we do this

sudo bash -c 'echo "exec /etc/X11/Xsession /usr/bin/xfce4-session" > /etc/chrome-remote-desktop-session'

Open Chrome Remote Desktop and click install via ssh

That's everything. Open up the connect page on remote desktop and its there. You can install chrome if you want to

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

sudo apt install ./google-chrome-stable_current_amd64.deb

Note: it will not stay turned on and almost everything will be deleted after session ends, and i know that the user home folder stays.

I recommend turning on dark mode in the XFCE settings and terminal settings
